00 —

Prerequisites

Before you begin, make sure you have these three things ready.

Mac with Terminal access
Claude Code installed & logged in (Pro recommended for longer outputs) · Internet connection
01 —

Install Homebrew + Node.js

Skip this step if you already have Node installed. Run node -v to check.

Install Homebrew
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
Add to PATH — Apple Silicon only
(echo; echo 'eval "$(/opt/homebrew/bin/brew shellenv)"') >> ~/.zprofile
eval "$(/opt/homebrew/bin/brew shellenv)"
Install Node.js + verify
brew install node

node -v    # e.g. v22.x or v24.x
npm -v     # e.g. 11.x
npx --version
If commands say "not found" after install — add this to ~/.zshrc and run source ~/.zshrc:

export PATH="/opt/homebrew/bin:$PATH"
02 —

Create the Remotion Project

This scaffolds your video project. Answer the prompts exactly as shown below.

npx create-video@latest
Critical prompt answers — wrong choices here make Claude struggle:

Template → Blank
Add Tailwind CSS? → Yes
Add agent skills? → Yes
Scope → Project
Enter project folder + install dependencies
cd my-video
npm install
Deprecation warnings and minor vulnerabilities are safe to ignore.
03 —

Increase Claude Code Output Limit

This prevents Claude from cutting off mid-generation on complex videos. Highly recommended.

Set for current session
export CLAUDE_CODE_MAX_OUTPUT_TOKENS=64000
Make permanent — add to ~/.zshrc then reload
echo 'export CLAUDE_CODE_MAX_OUTPUT_TOKENS=64000' >> ~/.zshrc
source ~/.zshrc
04 —

Launch Claude Code + Initialize

Run /init first — it creates a CLAUDE.md that makes Claude dramatically more accurate for your project.

claude
Inside Claude Code — run this first
/init
05 —

Paste the Main Prompt

Copy this prompt into Claude Code. Replace all [bracketed placeholders] with your own name, roles, and experience before pasting.

Claude Code prompt
Using the Remotion skill and best practices (including Tailwind for styling), create a professional 60-second video resume for [Your Name]. It should be modern motion graphics: minimalist, tech-forward, clean kinetic typography, subtle transitions (fades, scales, line reveals), dark background with blue/teal accents.

Duration: 60 seconds (55-65s ok). 1920x1080, 30fps.

Style: Dark mode (#0a0a1f bg), accents #00aaff/#00d4ff, white/gray text. Sans-serif fonts. Subtle glows/particles on key phrases. Smooth transitions.

Audio: Upbeat motivational instrumental (tech/innovation electronic, sync swells to highlights). Use <Audio src={staticFile("music.mp3")} volume={0.38} loop /> or suggest free Pixabay URL. No voiceover.

Structure with timed scenes:
- 0-10s: Intro – Large animated name "[Your Name]" + "[Your Title] | [Your Company]"
- 10-25s: Summary – Animate 2-3 sentences about your expertise, approach, and what drives you.
- 25-45s: Experience – Sequential animated bullets: [Role 1 + Company], [Role 2 + Company], [Role 3 + Company].
- 45-55s: Education & Skills – [Degree, School]. Skills tags: [Skill 1], [Skill 2], [Skill 3], [Skill 4], [Skill 5].
- 55-60s: Close – "[Your closing statement]" + LinkedIn / contact prompt.

Tone: Confident, professional, and true to your personal brand.

First: Detailed scene-by-scene plan with timestamps, text, animation ideas, Tailwind classes.
Then: Generate/update code (src/Composition.tsx, components). Use <Sequence>, spring(), Tailwind.

Iterate based on preview!
06 —

Preview Live

Open a new Terminal tab (Cmd+T) while Claude Code is still running in the other one.

npm run dev
Browser opens at localhost:3000. Refresh after each change Claude makes.
07 —

Add Background Music

Find a free instrumental on Pixabay Audio — search "corporate upbeat instrumental."

  1. Download an MP3 and rename it to music.mp3
  2. Drag it into your project's public/ folder
  3. Tell Claude the following:
Tell Claude Code
Use public/music.mp3 in the Audio component with volume 0.4 and loop
08 —

Render Final MP4

Before rendering, confirm your composition ID — it might differ from the default.

Confirm composition ID first
npx remotion compositions
Render to MP4
npx remotion render VideoResume out/resume.mp4 --codec=h264 --audio-codec=aac
Output lives at out/resume.mp4. Open with QuickTime or VLC.
09 —

Common Tweaks

Paste any of these directly into Claude Code to refine the result.

Increase intro name font to 180px and add drop shadow for readability
Shorten summary to 12s, stagger bullets more dramatically
Lower music volume to 0.3 + add 1s fade-in
Add subtle sparkle/glow effect on key highlight phrases
Make skills tags larger (text-5xl) and space them out
Use brighter teal accents (#00cfff) for better contrast
